Why would you want to write you code in a object-oriented programming language?

Object-oriented programming offers various advantages for example the reusability and maintainsability. It is also better to read. By organizing code into objects, you can easily build complex systems by breaking them down into smaller and separate pieces. it is also more organised and easier to read.

What's the difference between a class and an object?

A class is a blueprint or template for creating objects. The class defines properties and behaviors that objects will have. An object is a instance of a class. It is a something based on the specifications laid out in the class. In other words, a class is the description of a concept, while an object is a instance of that concept, with specific values for its properties
